## 📖 명언 앱

### ⚙️ 기능 구현 목록
### 🧾 입력
- 명령어를 입력받는다.
  - 지원 명령어: 등록, 목록, 삭제?id=<번호>, 수정?id=<번호>, 종료 
  - `ERROR` : 지원하지 않는 명령어 입력 시 안내 메시지를 출력한다.
- 명언 등록 시 입력 받는다.
  - 명언 내용과 작가를 입력받는다.   
  - `ERROR` : 빈 값 입력 시 예외 처리한다.

### 📝 등록
- 명언 등록 시 고유 번호를 자동 생성한다.
  - 등록 시마다 번호가 증가한다. 
  - 삭제된 번호는 재사용하지 않는다.
